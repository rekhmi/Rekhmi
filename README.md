# 🔬 Rekhmi AI - المساعد الذكي لمعامل التحاليل الطبية والكيميائيين

**Rekhmi AI** هو مساعد ذكي متطور مصمم خصيصاً لدعم الكيميائيين وأخصائيي التحاليل الطبية في إدارة وتسهيل سير العمل داخل المختبرات الطبية، وتقديم حلول أتمتة ذكية وموثوقة.

---

## 🌟 الميزات الرئيسية (Features)

* 🤖 **مساعد شات بوت ذكي:** للإجابة على الاستفسارات العلمية وتسهيل الإجراءات المختبرية.
* 🧪 **حاسبة التحاليل الطبية:** دعم الحسابات المعقدة والدقيقة (مثل حاسبة تحليل السائل المنوي بناءً على معايير منظمة الصحة العالمية WHO).(قيد التحسين)
* 📋 **تقنية الـ OCR الذكية:** قراءة وتحليل الروشتات والتقارير الطبية وتفريغ بياناتها بدقة لتوفير الوقت(قيد الإنشاء)
* ⚡ **أداء سريع وخفيف:** مبني بأحدث التقنيات ليعمل بسلاسة تامة كـ Web Application.

# 🔬 Rekhmi AI - Intelligent Assistant for Egyptian Medical Laboratory Chemists

**Rekhmi AI** is an advanced smart assistant specifically designed to support chemists and medical laboratory professionals in managing and optimizing laboratory workflows, providing intelligent and reliable automation solutions.

---

## 🌟 Key Features

* 🤖 **Smart Chatbot:** Answers scientific queries and streamlines laboratory procedures.
* 🧪 **Medical Lab Calculators:** Supports complex and precise calculations (such as the Semen Analysis Calculator based on WHO standards). *(Under Improvement)*
* 📋 **Smart OCR Technology:** Reads, analyzes, and extracts data from medical prescriptions and reports to save time. *(Under Construction)*
* ⚡ **Fast & Lightweight:** Built using modern technologies to run smoothly as a responsive Web Application.
